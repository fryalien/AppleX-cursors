**INSTALL ALL THESE**

```
yay -S --needed python-clickgen
```

```
sudo pacman -S --needed yarn
```

```
sudo pacman -S --needed nodejs npm
```

```
sudo npm install -g npm@11.9.0
```

```
npm install ful1e5/cbmp
```

```
yarn add ful1e5/cbmp
```
---

>Enter the repo

```
cd ~Public/AppleX-cursors/
```

---

>AppleX-Yellow #FFF000

```
npx cbmp -d "svg" -o "bitmaps/yellow" -bc "#FFF000" -oc "#000000"
```

>LinuX-Yellow

```
ctgen configs/x.build.toml -d "bitmaps/yellow" -n "MacOS-Yellow" -c "Yellow and Black macOS cursors."
```

>Win-Yellow

```
ctgen configs/win_lg.build.toml -d "bitmaps/yellow" -n "MacOS-L-Yellow" -c "Yellow and Black Win Large cursors."
```

```
ctgen configs/win_rg.build.toml -d "bitmaps/yellow" -n "MacOS-R-Yellow" -c "Yellow and Black Win Regular cursors."
```

```
ctgen configs/win_xl.build.toml -d "bitmaps/yellow" -n "MacOS-XL-Yellow" -c "Yellow and Black Win XL cursors."
```
